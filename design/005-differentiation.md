# Design Doc #005: 差异化 — 跟市面所有三国游戏不一样

## 问题
当前游戏就是一个标准的三国卡牌RPG——关卡、抽卡、升级、打Boss。
市面上这种游戏几百个，没有任何理由让人玩这个而不是玩别的。

## 市面三国游戏的套路（我们不要做的）
- ❌ 常规关卡推图
- ❌ 抽卡为核心
- ❌ 竞技场排名
- ❌ 纯数值碾压
= 所有三国手游的标配，没有差异

## 差异化方向（参考芒果庄园思路）

### 1. 🏯 三国·经营 (Kingdom Builder)
**你不只是打仗，你还要治理国家。**
- 战斗之间，你在经营自己的城池
- 建造：粮仓、兵营、铁匠铺、酒馆、书院
- 建筑产出资源（不是挂机数字，是可视化的城池成长）
- 城池外观随建筑升级变化（从小村落→城镇→都城）
- **差异点**：不是打完就走，你有一个"家"

### 2. 🗺️ 天命地图 (Destiny Map)
**不是线性关卡，是开放世界地图。**
- 三国大地图，你从一个点出发
- 可以选择进攻方向（先打荆州还是益州？）
- 不同路线遇到不同英雄和故事
- 占领城池后可以驻军、收税
- 地图上有随机事件（流民、商队、刺客）
- **差异点**：不是一条线推到底，是自己选择命运

### 3. 💬 英雄有"人格" (Hero Personality)
**英雄不是数值，是有性格的人。**
- 每个英雄有"心情"系统（开心/愤怒/低落/激动）
- 心情影响战斗表现（愤怒的张飞ATK+20%但可能失控攻击队友）
- 英雄之间会"聊天"——阵营内对话随机触发
- 做出不同的天命抉择会让某些英雄满意/不满（忠诚度）
- 忠诚度低的英雄可能叛逃！
- **差异点**：英雄是活的，不是棋子

### 4. ⚖️ 天命抉择有后果 (Choices Matter)
**不只是选A还是B，选择会改变整个游戏。**
- 当前的天命抉择只影响走哪条路
- 改成：选择影响后续所有章节
  - 选择火烧乌巢→曹操势力壮大→后续章节曹操系敌人更强
  - 选择正面决战→袁绍势力残留→后续出现袁绍复仇支线
- 结局不只一个，至少3种大结局
- 鼓励重复游玩看不同结局
- **差异点**：你的选择真正改变了世界

### 5. 📜 谋略系统 (Strategy Cards)
**不只是站桩互砍，战前部署决定胜负。**
- 战斗前选择"谋略"（类似昆特牌/卡牌机制）
- 火攻：敌方全体灼烧但我方也受影响
- 空城计：概率让敌方撤退（但失败则我方士气暴跌）
- 反间计：让一个敌方英雄这回合攻击队友
- 连环计：敌方全体锁定无法切换目标
- 每场战斗只能用2张谋略，选择很重要
- **差异点**：战前智慧 > 数值碾压

### 6. 🌦️ 天时地利 (Dynamic Battlefield)
**战场不是静态的，环境在变。**
- 已有地形和天气，但要更深入
- 战斗中天气可能变化（晴→雨→风→火借风势）
- 地形可破坏（火烧后森林变焦土）
- 时间流逝：白天/黑夜影响可见度
- 季节变化：冬季冻结河流可通行
- **差异点**：每场战斗都是独特的

## 最终定位
**三国·天命 = 经营建设 + 策略卡牌 + 叙事RPG + 开放路线**

不是打了就走的快餐，是一个你真正"活在"里面的三国世界。

## 实现优先级（从最大差异化开始）
1. ⭐⭐⭐ 天命地图 (替代线性关卡)
2. ⭐⭐⭐ 城池经营 (给玩家一个"家")
3. ⭐⭐ 谋略卡牌 (战前策略层)
4. ⭐⭐ 英雄人格 + 忠诚度
5. ⭐ 天命抉择后果链
6. ⭐ 动态战场
