# Design Doc #004: 一年可玩量 + 故事深度 + 难度平衡

## 问题
- 100关主线2-3天通关，没有长期可玩性
- 没有故事剧情，就是一关接一关打
- 难度要么秒杀要么碾压，没有中间态

## 方案：7大系统

### 1. 故事剧情系统 (Story Engine)
**每个关卡前后加对话/旁白**，不是干巴巴打完就走。
- 关卡开始前：对话场景（2-4句），交代背景
- 关键节点：动态剧情（天命抉择时的故事分支）
- Boss战前：Boss独白 + 主角回应
- 通关后：结果叙事 + 下一章预告
- **英雄羁绊故事**：收集到特定组合解锁专属剧情
  - 桃园三结义（刘关张）
  - 卧龙凤雏（诸葛亮+庞统）
  - 虎痴战神（许褚+典韦）
  - 吕貂之恋（吕布+貂蝉）

### 2. 自适应难度 (Adaptive Difficulty)
- 连胜3场 → 敌人+10%属性（逐步加压）
- 连败2场 → 敌人-15%属性（防止卡关）
- 维持50-70%胜率为最佳体验区
- 每章有推荐战力提示
- 支线可刷经验/金币/装备补强

### 3. 英雄羁绊系统 (Bond System)
- 英雄之间有"羁绊"关系（友好/敌对/暧昧/师徒）
- 同阵容放入有羁绊的英雄 → 触发羁绊技能/属性加成
- 羁绊故事需要通过使用这些英雄组合打关解锁
- **50+条羁绊线** = 长期收集动力

### 4. 周回内容 (Rotating Content)
| 周期 | 内容 | 奖励 |
|------|------|------|
| 每日 | 3个每日副本 + 3个每日任务 | 金币/经验/材料 |
| 每周 | 周Boss挑战（固定Boss，每周换） | 稀有装备/碎片 |
| 每两周 | 征讨战（限时活动关卡） | 限定英雄碎片 |
| 每月 | 月度锦标赛（竞技场赛季结算） | 排名奖励 |
| 每季 | 赛季通行证（新主题+新英雄） | 赛季限定 |

### 5. 无尽塔 (Endless Tower)
- 100层固定塔，每10层一个Boss
- 通关100层后解锁"深渊塔"（无限层，每5层难度跳一档）
- 排行榜记录最高层
- 每月重置前50层，保留进度奖励

### 6. 英雄养成深度
- 等级上限提升到100（当前可能较低）
- 突破系统：满级后用碎片突破，解锁新被动
- 装备锻造：不只是捡，可以合成+强化
- 天赋树已有，需要更多节点

### 7. 成就系统 (Achievements)
- 100+成就，分短期/中期/长期
- 短期：通关第1章、首次暴击、收集5个英雄
- 中期：通关精英难度、竞技场达到金牌、收集20个英雄
- 长期：全英雄满级、无尽塔100层、全羁绊解锁
- 成就奖励：宝石/限定头像框/称号

## 实现优先级
1. ⭐ 故事剧情系统（最直接提升体验）
2. ⭐ 自适应难度（解决太难/太简单）
3. 英雄羁绊系统（长期动力）
4. 成就系统（目标感）
5. 无尽塔完善
6. 周回内容丰富化

## 难度曲线设计
```
章节  推荐战力   敌人Scale   体验
1     500       0.3       教程，必胜
2     1000      0.5       开始需要搭配
3     2000      0.8       需要升级装备
4     3500      1.0       天赋树投入
5     5000      1.3       元素克制重要
6     7000      1.6       极限搭配
7     9000      1.8       需要羁绊加成
8     12000     2.0       需要突破英雄
9     15000     2.3       深渊级难度
10    20000     2.6       最终挑战
```
